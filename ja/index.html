📜 卒業後労働権 (2026-11-16改編基準) [公式]
  • 学士(L7)・修士(L9) → 卒業ビザ最大3年
  • GD(L7・学士先行必須) → 1年
  • L5–7 非学位正規課程(24週+) → 短期6ヶ月 (生涯1回)
  • L4以下 → なし
  • 学業中の労働は週25h ・ 学生ビザ $850 + 資金証明 年 $20,000 別途
💰 コスパランキング
  • ① 畜産 ~0 — 働きながら学ぶ
  • ② 会計 MPA 30.4k — 1.5年 → 卒業ビザ3年
  • ③ IT GD 26.4k — 1年 → 1年 (学士保有者)
  • ④ 教師 GradDip 28.7k~ — 1年 → GL T1
  • 対極: エンジニア学士 ~200k+
⚠️ 留学が罠の職種
  • 調理・園芸 — 卒業ビザ指定職業ではない
  • 課程を終えても労働権6ヶ月以下 — 学費26k+が無駄に
  • この2つはワーホリ現場参入が上位互換
🗣️ 英語の関門ランキング
  • 教師 — IELTS 7.0 全領域 (最高の壁)
  • 看護登録 — 7.0級 [推定]
  • 修士 6.5 ・ 学士 6.0 ・ 職業課程 5.0–5.5
  • 入学英語 ≠ 登録英語(看護・教師) — 別の関門

💵
到着バッファ 6,000+

WHVの資金証明は最低4,200ドル。最初4–6週の無収入 + 定住費2,000–3,500 + 次の家賃は別途準備。

⚡
初収入 2週以内

ホスピタリティ・シーズンジョブは比較的早い。成人最低賃金23.95ドル/時（税前）を下限として確認。履歴書は出国前に準備。

🏠
住居 = 貯蓄速度

ルームシェア 200–320 vs スタジオ 400+. シーズンジョブ食事提供なら貯蓄2倍 + 園芸3ヶ月 = WHV延長。

🧾
到着週にIRD

無料発行。雇用主にIR330の税コードを提出しない場合、45%にACC levyを加えたnon-notified税率が適用されます。

🏦
ビザ費分離積立

AEWV 1,540 ・ 永住権 6,450 — 申請2–3ヶ月前に別口座に。

🎓
留学現金フロー

学期中 25h(週 ~599)は生活費のみ相殺 — 学費は事前確保。
