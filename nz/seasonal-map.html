🌿 워홀 +3개월 연장 적격작업·시즌 지도

한국 국적 12개월 워홀 기준 · Working Holiday Extension Work Visa(WHE)와 시즌 일자리 위치를 한 화면에서 확인합니다.

3개월은 고정 88·90일이 아님

현재 워홀 중 적격 원예·포도 재배 업종에서 합계 최소 3개월. 여러 고용주·여러 지역·비연속 근무도 가능합니다.

현재 뉴질랜드 안에서 신청

현재 Working Holiday Visa를 보유하고 있어야 하며, WHE를 전에 받은 적이 없어야 합니다. 잡오퍼는 필요하지 않습니다.

증거를 보관

고용주 확인서·급여명세·세금 기록으로 3개월과 실제 작업을 입증합니다. 고용 상태의 기상 중단 기간은 포함될 수 있습니다.

현재 공식 안내: 체류·근로 3개월 추가 · 비용 NZD 800부터 · 신청의 80%가 3주 이내. 성인 최저임금은 2026-04-01부터 NZD 23.95/h이며 성과급도 실제 근로시간 기준 최저임금 이상이어야 합니다.
🗓️ 일할 달 선택

12개 주요 거점

월 선택 시 해당 달의 공식 캘린더 기반 기회가 강조됩니다. 실제 채용·날씨는 매년 변동.

어떤 일이 연장 실적으로 인정되나

판정작업설명
인정작물 심기·유지관리·수확·포장horticulture 또는 viticulture의 작물 자체를 대상으로 한 업무. 과수원·포도밭·팩하우스라도 실제 직무를 증명해야 합니다.
불인정종묘장(nursery)공식 운영지침에서 원예·포도 재배 연장 실적의 정의에서 제외합니다.
불인정식품 가공·제조주스·통조림·와인 제조·병입 등 가공·제조는 작물 포장과 다릅니다. 와이너리 근무라는 이유만으로 자동 인정되지 않습니다.
불인정낙농·축산·관광·일반 창고워홀 중 근무는 가능할 수 있어도 WHE의 horticulture/viticulture 3개월 실적에는 포함되지 않습니다.

WHE 연장과 SSE는 다른 비자

Working Holiday Extension Work Visa

  • 현재 워홀 보유자 + 이미 적격 작업 3개월
  • 최대 3개월 체류·근로 추가, 잡오퍼 불필요
  • 뉴질랜드 안에서 1회만 신청

Supplementary Seasonal Employment Work Visa

  • 워홀 자동 연장이 아닌 별도 시즌 고용 비자
  • 보통 visitor/student visa 보유자 대상
  • 승인된 SSE 고용주의 잡오퍼가 필요하며 최대 6개월
